# GFXTAB Product Roadmap & Specification

## Brand Identity
**Name:** GFXTAB
**Tagline:** Creative Marketplace & Design Request Platform
**Style:** Premium, Modern, Dark Mode

## Core User Flows
1. **Browse & Purchase:** Home -> Category -> Product Detail -> Payment -> Instant Download.
2. **Custom Request:** Home -> Request Form -> Admin Review -> Project Management.
3. **AI Assistance:** Chat Interface -> Requirement Analysis -> Template Recommendation.
4. **User Management:** Dashboard -> Orders -> Downloads -> Support.

## Screen List
1. **Marketplace Home:** Hero section, featured categories (Wedding, Social Media, Logos), trending assets, and "How it works".
2. **Product Details Page:** High-res preview, licensing info, asset specs (AI, PSD, etc.), and "Add to Cart".
3. **Custom Design Request Portal:** Multi-step form for project details, budget, and file uploads.
4. **User Dashboard:** Centralized hub for downloads, active project status, and communication.
5. **AI Design Assistant (Overlay/Module):** Interactive sidebar for guided discovery.

## Technical Highlights
- Next.js & Tailwind CSS for performance.
- Framer Motion for smooth transitions.
- Razorpay/Stripe integration for global/local payments.
